# 차량 파손 부위 감지 및 위치 파악

## 개요
본 프로젝트는 차량 파손부위 감지하고 위치를 파악하는 기능을 구현합니다. YOLO모델을 기반으로 하며, Roboflow의 Car damage_2 Computer Vision Project by Socar 데이터셋을 사용했습니다. 


## 주요 기능
- 데이터에 맞는 모델의 이미지로 학습 결과 확인


## 사용 모델
YOLOv8, YOLOv5


**학습 데이터:**
* **데이터셋:** 뉴스 기사 데이터셋을 사용했습니다.
